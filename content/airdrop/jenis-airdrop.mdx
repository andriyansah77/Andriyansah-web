## Overview

Tidak semua airdrop sama. Setiap jenis punya cara kerja, effort, dan potensi reward yang berbeda. Pahami semuanya supaya kamu bisa pilih yang paling cocok.

## 1. Retroactive Airdrop

**Apa:** Token diberikan ke user yang sudah pakai produk sebelum token launch.

**Contoh terkenal:** Uniswap ($UNI), Arbitrum ($ARB), Optimism ($OP)

**Cara ikut:**
- Pakai produk sedini mungkin
- Lakukan transaksi nyata (swap, bridge, provide liquidity)
- Jangan cuma 1x — interaksi rutin lebih baik

**Kelebihan:** Potensi reward paling besar
**Kekurangan:** Tidak ada jaminan, butuh waktu lama

## 2. Task-Based Airdrop

**Apa:** Selesaikan task tertentu (follow, retweet, join discord, quiz) untuk dapat token.

**Cara ikut:**
- Cek platform seperti Galxe, Layer3, Zealy
- Selesaikan task yang diminta
- Biasanya perlu connect wallet

**Kelebihan:** Jelas apa yang harus dilakukan
**Kekurangan:** Reward biasanya lebih kecil, kompetisi tinggi

## 3. Testnet Airdrop

**Apa:** Ikut testing jaringan blockchain sebelum mainnet launch.

**Cara ikut:**
- Claim faucet (testnet token gratis)
- Lakukan transaksi di testnet
- Report bug kalau menemukan
- Ikuti instruksi di docs project

**Kelebihan:** Biasanya tanpa modal, project menghargai early tester
**Kekurangan:** Teknis, butuh waktu, tidak semua testnet berujung airdrop

## 4. Social / Campaign Airdrop

**Apa:** Reward untuk aktivitas sosial — share konten, buat thread, engagement.

**Cara ikut:**
- Buat konten original tentang project
- Ikut campaign resmi
- Engage di community (bukan spam)

**Kelebihan:** Cocok untuk yang suka bikin konten
**Kekurangan:** Subjektif, tergantung penilaian project

## 5. Community Airdrop

**Apa:** Reward untuk kontributor komunitas — moderator, helper, active member.

**Cara ikut:**
- Jadi member aktif di Discord/Telegram project
- Bantu jawab pertanyaan member lain
- Lamar jadi moderator atau ambassador
- Kontribusi nyata ke komunitas

**Kelebihan:** Membangun network dan reputasi
**Kekurangan:** Butuh waktu konsisten, tidak semua project menghargai ini

## 6. NFT-Based Airdrop

**Apa:** Punya NFT tertentu → dapat airdrop token.

**Cara ikut:**
- Mint atau beli NFT dari project tertentu
- Hold di wallet
- Biasanya ada snapshot date

**Kelebihan:** Pasif — cukup hold
**Kekurangan:** Butuh modal untuk beli/mint, risiko NFT turun harga

## 7. DApps / Protocol Airdrop

**Apa:** Pakai dApps di ecosystem tertentu — DEX, lending, bridge, dll.

**Cara ikut:**
- Gunakan protocol secara rutin
- Diversifikasi: swap, lend, bridge, stake
- Fokus di ecosystem baru yang belum ada token

**Kelebihan:** Paling natural — kamu pakai produk sungguhan
**Kekurangan:** Kadang butuh modal untuk gas dan transaksi

## 8. Telegram Bot / Mini App

**Apa:** Bot Telegram atau mini app yang memberikan reward berdasarkan engagement.

**Cara ikut:**
- Join bot/mini app
- Selesaikan daily task
- Invite friends (biasanya ada referral bonus)
- Stay active

**Kelebihan:** Mudah, bisa dilakukan dari HP
**Kekurangan:** Banyak yang tidak deliver, reward kecil, banyak scam

## Strategi: jangan pilih satu, diversifikasi

Jangan cuma fokus di satu jenis. Kombinasi yang bagus:

- **Retroactive** (2-3 project utama yang kamu pakai rutin)
- **Task-based** (sambil jalan, selesaikan quest di Galxe/Layer3)
- **Testnet** (kalau ada project menarik yang buka testnet)
- **Content** (kalau kamu bisa bikin konten, ini bonus besar)
